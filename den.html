Streetwear Den Mobile App

In a prior Humber College project, we used our UI and UX skills to design a brand and a mobile app, which is how the streetwear den mobile app came to be. I made the decision to develop a mobile fashion app where you can browse and buy used designer clothing and shoes. Using Adobe XD, I was able to design this application and create a user flow journey prototype. During this journey, there was a three-second animation screen that led to the app's login page. Once you log in, you may access the scrollable homepage. Then going to a section dedicated to categories, where one can select an item by clicking on an option that takes one to the shop page which they can add to their basket and checkout
